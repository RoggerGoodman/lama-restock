Upload Delivery Document (DDT)

What This Does
  1. Upload the DDT (Documento Di Trasporto) from your supplier
  2. System extracts product codes and quantities
  3. Stock levels are automatically updated for all products
  4. Report shows what was added and any issues
Requirements
  • File must be a valid PDF
  • PDF must contain product codes in format: XXXXX.XX
  • Each line should show: CODE, description, PZ, packages, pieces/package, QUANTITY
  • Only products already in the database will be updated
